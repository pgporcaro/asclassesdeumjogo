# 🎮 Desafio: Escrevendo as Classes de um Jogo

Este repositório contém a resolução do terceiro desafio de lógica de programação do bootcamp da DIO. O objetivo principal foi aplicar conceitos de **Programação Orientada a Objetos (POO)** utilizando a linguagem JavaScript.

## 📝 Objetivo do Desafio

Criar uma classe genérica que represente um herói de uma aventura e que possua as seguintes propriedades:

- **Nome**
- **Idade**
- **Tipo** (ex: guerreiro, mago, monge, ninja)

Além disso, a classe deve ter um método chamado `atacar` que deve atender aos seguintes requisitos:

- Exibir a mensagem: `"o {tipo} atacou usando {ataque}"`
- O `{tipo}` deve ser o tipo definido na classe.
- O `{ataque}` deve variar conforme o tipo, seguindo a tabela abaixo:

| Tipo | Ataque |
| :--- | :--- |
| Mago | Magia |
| Guerreiro | Espada |
| Monge | Artes Marciais |
| Ninja | Shuriken |

---

## 🚀 Tecnologias Utilizadas

* **JavaScript (ES6+)**: Linguagem principal para a lógica.
* **Node.js**: Para execução do script no terminal.
* **Classes e Objetos**: Estrutura fundamental do projeto.
* **Estruturas de Decisão**: Implementação de lógica com `switch` ou `if/else`.
